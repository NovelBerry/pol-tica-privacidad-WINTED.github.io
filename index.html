Términos de Servicio y Política de Privacidad de WINTED

1. Términos de Servicio (TdS)

Al interactuar con WINTED, usted acepta estar legalmente vinculado a los siguientes Términos de Servicio. El uso continuado del bot constituye su consentimiento a estos términos.

1.1 Uso del Servicio

WINTED está diseñado para ser una herramienta de utilidad, moderación y entretenimiento. Cualquier uso indebido, incluyendo, pero no limitado a, el abuso de comandos o la explotación de fallos, está estrictamente prohibido y será sancionado.

1.2 Moderación

Los comandos de moderación son herramientas exclusivas para el personal autorizado del servidor (staff). El uso de estos comandos debe ser responsable, justo y alineado con las reglas del servidor.

1.3 Propiedad y Datos

Los datos generados por el bot, como el saldo de la economía y los registros de advertencias, se consideran propiedad del bot y se almacenan exclusivamente para el funcionamiento del servicio. Estos datos no son transferibles y pueden ser modificados o eliminados sin previo aviso, en caso de ser necesario.

1.4 Limitación de Responsabilidad

No nos hacemos responsables por la pérdida de datos virtuales, la interrupción del servicio, o cualquier daño derivado del uso de WINTED. Usted utiliza el servicio bajo su propio riesgo.

2. Política de Privacidad (PP)

2.1 Recopilación de Información

WINTED recopila y almacena la siguiente información con el único propósito de proveer sus funcionalidades:

  • Identificador de Usuario (ID): Se utiliza para asociar los datos de economía y las sanciones con un usuario específico de Discord.
  • Datos de Economía: Incluye el dinero virtual (cash y bank) y el progreso de misiones.
  • Datos de Moderación: Incluye registros de advertencias.

2.2 Uso de la Información

La información personal no se comparte, vende, ni se distribuye a terceros. Los datos se utilizan exclusivamente para las funciones internas del bot, como el sistema de economía y moderación, garantizando una experiencia de usuario coherente.

2.3 Almacenamiento y Seguridad de Datos

La información se almacena localmente en archivos de texto protegidos. Hacemos esfuerzos razonables para proteger los datos contra accesos no autorizados, pero no podemos garantizar una seguridad absoluta. El acceso a los datos está estrictamente limitado al propietario del bot.

2.4 Acceso y Eliminación de Datos

Los usuarios pueden solicitar información sobre los datos que el bot tiene de ellos. Para solicitar la eliminación de su información personal, póngase en contacto con el propietario o administrador del servidor. La eliminación de datos es permanente y no reversible.
